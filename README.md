RFID Library Management System

##mProject Overview

An automated library management system using RFID technology to handle book issuing, returning, and student entry tracking.
The system integrates hardware (RFID + NodeMCU/Arduino) with a backend server and web dashboard for real-time data management.

---

## Features
- Automated book issue and return using RFID cards
- Student entry tracking system
- Real-time data updates and logging
- Web dashboard for monitoring records
- Email notifications for issued/returned books

---

## Tech Stack
- **Hardware:** RFID Module, Arduino Uno / NodeMCU
- **Backend:** Node.js
- **Frontend:** HTML, CSS, JavaScript
- **Database:** JSON-based storage
- **Other:** Python (RFID logging)
